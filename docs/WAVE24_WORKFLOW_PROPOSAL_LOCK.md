## Honest Caveats

- Wave 24 creates workflow proposal evidence only. It does not create workflow runs.
- A reviewed workflow proposal still does not authorize execution.
- The deterministic builder creates one stage per plan step. Real-world decomposition may need LLM assistance.
- Tool intents are descriptive categories (e.g. "context-observation"), not registered tool names.
- ApplyChange stage kind is defined but not generated by the current builder (no TaskPlanStepKind maps to it).
- No workflow versioning or branching — a new intent creates a new proposal.
- No concurrent proposal editing — single-user assumption holds.
